# 介绍

Gin 是一个用 Go 语言 (Golang) 编写的 HTTP web 框架。它具有类似 Martini 的 API，但性能比 Martini 快 40 倍。如果你需要极好的性能，使用 Gin 吧。

Create，First，Find，Take，Save，Update，Delete，Scan，Row，Rows…
